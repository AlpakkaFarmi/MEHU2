# MEHU2

## Introduction

In the modern world full of wires and ports, it's easy to make a mistake.
One wrong move, one slip, and your high secure environment is in public network.

To remedy this, MEHU2 aims to establish the golden standard for preventing offline-environments to communicate with Internet.

## Requirements

**Candidate is evaluated based on following requirements**

1. Program has to recognize that communication path to Internet is possible
2. After recognizing Internet connectivity, program needs to shutdown the system as soon as possible
3. Program must be executable in Linux, Windows and MacOS

## Instructions to apply

** Make PR for this repostory to submit a new candidate program for MEHU2** 

### PR has to include
* Source code
* Compilation instructions
